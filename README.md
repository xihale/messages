# Messages
## Take some messages

使用方法: 

1. 获取源码

   ```bash
   git clone https://gitee.com/xihale/messages.git
   ```
   
2. 配置参数
   首先打开`config.php`
   `server`->`数据库所在服务器的ip`
   `username`->`数据库用户名`
   `dbname`->`数据库名称`
   `password`->`用户密码`

3. 创建数据表

   访问`init.php`即可(学习数据库的时候写的,所以别问我为什么没有写完整)
